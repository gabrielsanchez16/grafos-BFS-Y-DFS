# Implementación de BFS y DFS en Grafos

Este proyecto implementa los algoritmos de búsqueda **Breadth First Search (BFS)** y **Depth First Search (DFS)** en Python utilizando un grafo pequeño como ejemplo.

El objetivo es comparar el comportamiento de ambos algoritmos en términos de:

- Camino encontrado
- Uso de memoria

## Descripción del proyecto

Los grafos son estructuras de datos ampliamente utilizadas en informática para representar relaciones entre elementos. Dos de los algoritmos más importantes para recorrer grafos son:

- **BFS (Breadth First Search)**: explora el grafo por niveles.
- **DFS (Depth First Search)**: explora el grafo en profundidad antes de retroceder.

En este proyecto se implementan ambos algoritmos en Python y se aplican sobre un grafo pequeño generado con la librería **NetworkX**.

## Tecnologías utilizadas

- Python
- Google Colab
- NetworkX
- Matplotlib

## Estructura del grafo

El grafo utilizado contiene varios nodos conectados mediante aristas. Se representa visualmente para facilitar la comprensión del recorrido de los algoritmos.

## Funcionamiento

El programa realiza las siguientes acciones:

1. Crea un grafo de ejemplo.
2. Visualiza el grafo utilizando NetworkX.
3. Implementa los algoritmos BFS y DFS.
4. Busca un camino entre dos nodos.
5. Compara el uso de memoria de cada algoritmo.

## Resultados

Los resultados muestran:

- El camino encontrado por cada algoritmo.
- La cantidad de memoria utilizada durante la ejecución.

Esto permite analizar las diferencias entre BFS y DFS en términos de eficiencia y comportamiento.
